# 备课 PPT 工作流

## 1. 输入材料

常见输入包括：

- 课程 PPT
- 论文题名、摘要或自写梳理
- 案例材料
- 教学大纲

真实课程资料通常不进入开源仓库。开源示例应使用脱敏材料或重新撰写的示例文本。

## 2. 风格继承

先观察课程 PPT 的以下特征：

- 页面比例
- 主色与强调色
- 标题位置
- 页脚线或装饰条
- 正文密度
- 是否使用卡片、流程图、左右分栏

## 3. 内容压缩

把长文本转换为可讲授的逻辑结构：

- 背景事实
- 核心概念
- 作用机制
- 风险边界
- 政策或实践含义

## 4. 版式检查

生成 PPT 后至少检查：

- 页数是否符合要求
- 图片数量是否符合要求
- 所有对象是否在页面范围内
- 文字是否可能溢出文本框
- 是否存在覆盖、错位或页脚遮挡
