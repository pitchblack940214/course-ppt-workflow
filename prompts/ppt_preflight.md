# PPT Pre-flight Prompt

在生成 PPT 前，先完成以下检查。

## Contract

- 用途：案例导入 / 论文精读 / 课程补充 / 组会汇报
- 受众：本科生 / 研究生 / 教师 / 评审
- 页数：2-4 页 / 5-8 页 / 用户指定
- 模板：课程 PPT / 答辩 PPT / 通用讲义风
- 输出：可编辑 `.pptx`
- 校徽：不需要 / 从校徽库匹配 / 从模板候选区域裁剪
- 图片：不需要 / 仅辅助 / 必须包含原文图表

## Materials Status

- PPT 模板是否可读取
- 论文或案例文本是否可读取
- 是否已有 PPT 大纲
- 是否需要提取图片或图表
- 是否需要校徽或学校标识
- 输出目录是否明确

## Success Criteria

- 逻辑清楚，能直接讲
- 继承模板风格
- 文字不超框
- 不使用整页截图
- 生成后通过页数、图片数和对象边界检查
