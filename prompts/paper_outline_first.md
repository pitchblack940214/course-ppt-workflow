# Paper Outline First Prompt

请先不要生成 PPT，先把论文整理成 PPT 大纲。

## 输出要求

1. 判断用途：课堂导入 / 论文精读 / 课程补充 / 组会汇报。
2. 判断建议页数：简要解读 2-4 页，组会精读 5-8 页。
3. 提炼 3-5 个课堂抓手或汇报抓手。
4. 按页列出：
   - 页标题
   - 核心结论
   - 关键要点
   - 建议图示类型
   - 是否需要原文图表
5. 最后给出一页讨论题或延伸问题。

## 默认论文精读结构

1. 题名与核心判断
2. 研究问题与理论背景
3. 结论概览
4. 机制链条或分析框架
5. 关键证据或案例事实
6. 风险边界与政策启发
7. 讨论题
